# 実効再生産数は数値目標として微妙
[5/1のスッキリ](https://www.j-cast.com/tv/2020/05/01385311.html?p=all)で皆さん知りたがっていた実効再生産数。

ドイツが毎日公表しているとかで謎の信頼を得ているこの数字ですが、日本の専門家達はこの数字より`倍化時間`が指標として使えると提言内で示唆しています。

実効再生産数が分かると確かに、感染者が急増しているのか減っているのかは一目瞭然です。
